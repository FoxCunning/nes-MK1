# Mortal Kombat for the NES
This is a hack of the Hummer Team's bootleg "Mortal Kombat 2" cartridge (which was actually a demake of Mortal Kombat 1) for the NES.

- Improved graphics
- New sound engine
- Improved music and sound effects
- Bug fixes
